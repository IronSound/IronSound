# IronSound

User login: 

  route = `POST`, path = `"/login"`, data = `User` object (contains Strings `name` & `password`) 
  
User logout:

  route = `POST`, path = `"/logout"` 

Show songs:  

  route = `GET`, path = `"/songs"`
  
Add a song: 

  route = `POST`, path = `"/songs"`, data = `int trackId`
